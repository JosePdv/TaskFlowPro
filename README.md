# Task Flow Pro

## Sistema estilo

- Trello / Jira simplificado.
- Multiusuário.
- Multi-organização.
  
##

### Funcionalidades

#### Backend

- Autenticação JWT;
- Multi-tenant (usuários com organizações);
- CRUD de:
 Projetos;
 Tarefas;
 Comentários
- Permissões (admin, user)
- Logs de atividade
- Paginação + filtros;
  
#### Frontend

- Dashboard moderno;
- Drag and Drop de Tarefas;
- Estados globais (Zustand ou Context);
- Consumo da API real;

## Diferenciais

- Arquitetura em camadas (service, repository)
- Validação robusta
- Documentação Swagger automática
- Testes básicos
